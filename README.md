# 🎓 Student Performance Predictor (Advanced ML Project)

## 📌 Project Overview
This project predicts student academic performance using Machine Learning models. 
It compares multiple algorithms and selects the best performing model using hyperparameter tuning and cross-validation.

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## 🤖 Machine Learning Models
- Linear Regression
- Random Forest Classifier
- Hyperparameter Tuning using GridSearchCV
- Cross Validation

## 📊 Features
- Feature Engineering
- Model Comparison
- Confusion Matrix Visualization
- Feature Importance Analysis
- Interactive Animated Performance Dashboard

## 📈 Model Performance
- Cross Validation Accuracy
- Training vs Testing Accuracy
- Tuned Random Forest Accuracy

## 🚀 How to Run
1. Install dependencies:
   pip install pandas numpy matplotlib scikit-learn
2. Run:
   python student_performance_predictor.py

## 🎯 Outcome
The tuned Random Forest model achieved the highest prediction accuracy.
